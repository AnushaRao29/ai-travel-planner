# AI Travel Planner ✈️

Generate personalized day-by-day travel itineraries instantly 
using AI. Enter a destination and number of days — get a 
complete itinerary in seconds.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- AI: Groq API (Mixtral)
- Deployment: Netlify + Render

## Features
- AI-generated itineraries by destination and duration
- Streaming response (real-time generation)
- Clean day-by-day card layout
